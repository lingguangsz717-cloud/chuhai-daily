# 出海早班车 · Cross-Border Daily

AI驱动的跨境电商日报。覆盖美区、东南亚、日韩、欧洲四大市场，每个工作日早上8点更新。

## 访问

https://你的用户名.github.io/chuhai-daily/

## 目录

- `index.html` — 单文件日报（数据内嵌，双击即看）
- `data/` — 每日数据存档（JSON格式）
- `generate_daily.py` — 每日数据生成脚本

## 数据来源

TikTok Trending · Amazon · Kalodata · FastMoss · 1688 · Google Trends · 各国央行/平台官方公告
